IdeaNaverAnalytics

== Description ==

네이버 애널리틱스 플러그인입니다. 애널리틱스 키를 발급받으시고 키를 넣으시면 애널리틱스 코드가 워드프레스에 반영되어 등록됩니다.

== Installation ==

1. 플러그인 활성화
2. 네이버 애널리틱스 키 발급
3. 워드프레스 관리자화면 ) 설정 -> NaverAnalytics 에서 키 입력.
